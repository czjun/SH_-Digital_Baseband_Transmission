# 数字通信系统仿真项目

## 项目概述

本项目是一个基于MATLAB的数字通信系统仿真平台，实现了完整的数字通信链路仿真，包括信源生成、信道编码、调制、滤波、噪声添加、解调、判决和误码率分析等功能。

## 主要功能

### 1. 信源生成
- 支持二进制数据流生成（0/1或1/-1）
- 可配置信源长度和数据类型

### 2. 信道编码
- AMI（Alternate Mark Inversion）编码
- 支持不同编码方案的性能对比

### 3. 调制方案
- BPSK（Binary Phase Shift Keying）
- QPSK（Quadrature Phase Shift Keying）
- 8PSK（8-Phase Shift Keying，格雷映射）

### 4. 滤波处理
- 升余弦滚降滤波器
- 支持不同滚降系数的性能分析
- 发送和接收滤波器

### 5. 信道仿真
- 加性高斯白噪声（AWGN）信道
- 可配置信噪比（SNR）

### 6. 信号检测
- 采样和判决
- 码型转换
- 误码率计算

- 显示接收信号的星座图

- 显示信号的眼图
- 分析不同滚降系数对眼图的影响

- 误码率（BER）vs 信噪比（SNR）曲线
- 理论值与仿真值对比
- 不同调制方式的性能对比

### 5. 滚降系数分析
- 分析不同滚降系数（0.1, 0.3, 0.5, 0.7）对系统性能的影响
- 眼图质量分析
- 误码率性能分析

### 6. 信道编码分析
- 对比使用信道编码前后的性能
- 分析编码增益

### 7. 多调制方式对比
- BPSK、QPSK、8PSK性能对比
- 格雷映射的8PSK实现
- 不同调制方式的误码率分析

## 使用说明

### 运行环境
- MATLAB R2016b或更高版本
- 需要Signal Processing Toolbox

### 运行步骤

#### 快速开始
1. 打开MATLAB
2. 将项目文件添加到MATLAB路径
3. 运行 `simple_test.m` 进行基本功能测试
4. 运行 `main_enhanced.m` 进行完整分析

#### 故障排除
如果遇到错误，请先运行 `simple_test.m` 验证基本功能

#### 详细分析
- `rolloff_analysis.m`: 分析不同滚降系数的影响
- `modulation_comparison.m`: 对比不同调制方式的性能
- `main_enhanced.m`: 完整系统仿真和分析

### 参数配置
- `type`: 信源类型（1: 0/1, 2: 1/-1）
- `a`: 滚降系数（0.1-0.9）
- `L`: 信源长度
- `SNR`: 信噪比范围
- `modulation_types`: 调制方式（'BPSK', 'QPSK', '8PSK'）
- `coding_types`: 编码方式（'none', 'repeat', 'convolutional'）

## 性能要求

- 误比特率要求达到1×10^-5以下
- 支持多种调制方式
- 完整的信道编码分析
- 详细的性能图表
